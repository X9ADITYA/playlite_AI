# 🎥 PlayLite AI

> **The AI video workspace for people who want answers, not just playback.**

PlayLite AI transforms video libraries into an interactive learning platform powered by AI. Upload videos once and instantly generate transcripts, summaries, timestamps, and ask questions directly about the content.

![PlayLite AI Banner](./public/banner.png)

---

## ✨ Features

### 📄 AI Summaries
- Automatic video summarization
- Key takeaways
- Bullet-point notes
- Important timestamps

### 💬 Ask the Video
- Chat with your uploaded videos
- Transcript-based Question & Answer
- Context-aware responses
- Jump to relevant moments instantly

### 📝 Video Transcription
- Fast and accurate speech-to-text
- Searchable transcripts
- Timestamp support

### 🎨 Creator Tools
Generate content directly from your videos:
- Video Titles
- Descriptions
- Thumbnail Ideas
- Social Media Captions

### 📚 Personal Learning Workspace
- Upload once
- Learn forever
- Organize your video library
- AI-powered recommendations

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/yourusername/playlite-ai.git
```

Go into the project directory

```bash
cd playlite-ai
```

Install dependencies

```bash
npm install
```

Run frontend

```bash
npm run dev
```

Run backend

```bash
npm run server
```

---

## 🔑 Environment Variables

Create a `.env` file inside the server folder.

```env
PORT=5000

MONGO_URI=your_mongodb_uri

JWT_SECRET=your_secret

OPENAI_API_KEY=your_api_key

GEMINI_API_KEY=your_api_key
```
